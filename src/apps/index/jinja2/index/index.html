What is Colorin?

Colorin is a fabulous app for analyzing Instagram photo palette and finding the most suitable ones for posting.

How it finds similar photo?

The process for searching images similar in color is implemented using k-means, arrays that expand each color of the palette and the matching algorithm

What also can do this application?

App can suggest suitable emoji pics, download good for posting images in ZIP, upload multiply files, update information.
